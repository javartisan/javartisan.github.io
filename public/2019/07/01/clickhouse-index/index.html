Clickhouse中的arrayjoin函数

通常SQL函数都是对一些rows进行一些map操作或者聚合操作，这些操作并不会导致行数的增加；然而arrayjoin是将一些rows转化为更多的rows，是一个扩增的过程。官方文档描述：The ‘arrayJoin’ function takes each row and generates a set of rows (unfold).

查询结果：

dst name
1 Hello Clickhouse
2 Hello Clickhouse
3 Hello Clickhouse

由于name是一个常量，arrayjoin数组长度为3，因此扩增为3行。
